# AI-Chef 智能食谱生成器 🧑‍🍳

一个面向全球用户的 AI 驱动菜谱生成工具，帮你快速发现、定制和分享美味食谱。

## ✨ 核心功能
- **多语言支持**：支持英语、西班牙语、法语等多种语言，服务全球美食爱好者
- **丰富菜系覆盖**：涵盖中餐、日料、美式、意式等主流国际菜系
- **智能生成**：基于 AI 快速生成步骤清晰、营养均衡的食谱
- **响应式设计**：适配手机、平板和桌面端，随时随地查看菜谱

## 🚀 快速开始
访问我们的网站即可开始使用：
- 域名：`taropai.com`
- 直接在浏览器中打开，无需下载安装

## 📝 使用说明
1.  选择你偏好的语言和菜系
2.  输入食材或口味偏好，点击生成
3.  查看完整的食材清单、烹饪步骤和营养信息
4.  保存或分享你喜欢的食谱

## 🛠️ 技术栈
- 前端：纯 HTML + CSS + JavaScript
- 部署：GitHub Pages + Cloudflare Pages
- AI 能力：集成第三方大模型 API
